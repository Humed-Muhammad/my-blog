You created the swap file to help you increase the performance of your computer, but now you want to remove it. Don't worry, in this short article, we will show you the different ways you can remove the swap file from your computer.

## First what is a swap file

I am not going to bore you with a huge paragraph because if you didn't know what it was, you never came to this site in the first place.

**swapfile** is a system file that creates temporary storage space on a solid-state drive or hard disk when the system runs low on memory.

know you know what it is lets list some of the ways we can remove this swapfile.

## How to remove swapfile

### 1. By Directly deleting the file

- First turn off the swapfile, open your terminal and type the command below.

```bash
sudo swapoff
```

- Then You can find the file in your root computer directory.

### 2. Using the terminal on the fly

- Open your terminal and type the command below.

```bash
sudo swapoff /swapfile/to/be/deleted

```

### or

```bash
sudo swapoff
sudo rm -f /swapfile
```

## Finaly

Finaly comment the entry in **/etc/fstab** file

- Open your terminal and type the command below.

```bash
sudo nano /etc/fstab
```

- Then comment out the **/swapfile swap swap defaults 0 0** line by placing **#** in front of it.
