## First check for existing ssh key

1. Open terminal and type:

```bash
ls -al ~/.ssh
```

This will list ssh keys if available:

<Note>
  **Note** Below are the supported file name for GitHub
  <ui>
    <li>id_rsa.pub</li>
    <li>id_ecdsa.pub</li>
    <li>id_ed25519.pub</li>
  </ui>
  If you do not have an existing SSH key pair in the default location. You can create a new SSH key
  pair by following [this](how-to-create-ssh-key) tutorial.
</Note>

2. Copy the SSH public key:

```bash
cat ~/.ssh/id_ed25519.pub
```

Then select and copy the contents of the id_ed25519.pub file

3. In the upper-right corner of any page, click your profile photo, then click **Settings**.
4. In the "Access" section of the sidebar, click SSH and GPG keys.
5. In the title field add the any name you want for your new ssh key.
6. Paste the SSH public key in the **Key** field.
7. Finally click **Add SSH key** button.
