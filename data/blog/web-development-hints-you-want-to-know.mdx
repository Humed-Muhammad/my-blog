1. #### CSS the :where pseudo selector

   The Css pseudo-class function where() accepts a selector list as an argument and selects any element that can be selected by one of the selectors in that list.

   - The below example is in nextjs but it would work on any css file.

2. #### Spellcheck attribute

The spellcheck attribute specifies whether an element's spelling can be checked.
It can be used in text inputs, textareas, and elements with contenteditable="true" attributes.

```html
<input spellcheck="true" type="text" />
```

4. #### Attribute selectors
   Did you know there are numerous ways to use attribute selectors in CSS?
   Take a look at this:

```css
div[class='cont'] {
  /* this check if the attribute is "cont" */
}
div[class*='cont'] {
  /* this check if the attribute contain "cont" */
}
div[class^='cont'] {
  /* this check if the attribute start with "cont" */
}
div[class$='cont'] {
  /* this check if the attribute end with "cont" */
}
div[class|='cont'] {
  /* this check if the attribute is "cont" or start with "cont" */
}
```

<Note>
  **Note:** The above selectors are all case sensetive if you want to make it case insensitive add
  `i` before the end of the closing braket `[class*='cont' i]`
</Note>

5. #### The Object fit attributes
   The object-fit CSS property sets how the content of a replaced element, such as an `<img>` or `<video>` , should be resized to fit its container.

```css
object-fit: contain;
object-fit: cover;
object-fit: fill;
object-fit: none;
object-fit: scale-down;
```

6. #### Console.time
   You can start a timer with console.time() to see how long an operation takes. In addition, you can use the timeLog method between the start and end of the timer. Here's a simple example:

```js
const runTask = (value) => {
  console.time(`for ${value}`)
  let result = 0
  for (let i = 0; i < value; i++) {
    result += i
  }
  console.timeEnd(`for ${value}`)
  return result
}
```
