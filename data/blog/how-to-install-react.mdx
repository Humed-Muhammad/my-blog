## Introduction

React is a free and open-source front-end JavaScript library for creating UI components-based user interfaces.

- Today we are going to see how to install react both on linux and windows operating systems. Installing react is simple and the same for both operating systems.

The difference lies in the installation of Nodejs.

#### Without further ado, let get started.

## How to install react on linux

- Follow the instructions below to install react on **Linux**

#### 1. Check if Nodejs is installed

```bash
node -v
```

- Output:

```bash
v18.8.0 or similar depending on your node version.
```

- If the output is not the same as the above, then you should install Nodejs to do so follow [this article](how-to-install-nodejs)

#### 2. Installing react on linux

- Now days there are several different ways to install react but the most popular one is to use **create-react-app** package developed by the facebook itself.

- Open your terminal ( Ctrl + alt + t )

- Then run the following command to install react:

```bash
npx create-react-app my-app
```

- Where **my-app** is the name of the folder that contains the react application, you can change the name to you own preferences.
