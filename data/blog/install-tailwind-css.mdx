## Introduction

Tailwind css is a utility-first CSS framework packed with classes like flex, pt-4, text-center and rotate-90 that can be composed to build any design, directly in your markup.

[For react follow this](how-to-install-tailwindcss-in-react)

## How to install tailwindcss for plain html projects

The Tailwind CLI tool is the quickest and easiest way to get started with Tailwind CSS from scratch.

1. Install Tailwind CSS via npm

```bash
npm install -D tailwindcss
```

Then run:

```bash
npx tailwindcss init
```

2. Configure your template path in `tailwind.config.js` like below:

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ['./src/**/*.{html,js}'],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

3. Add the Tailwind directives to your CSS (e.g. in index.css)

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

4. Start the Tailwind CLI build process

```bash
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```

5. Start using Tailwind

Add your compiled CSS file to the `<head>` and start using Tailwind's utility classes to style your content.

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <!-- link you compaild css like this -->
    <link href="/dist/output.css" rel="stylesheet" />
  </head>
  <body>
    <h1 class="text-3xl font-bold underline">Welcome, Hello world!</h1>
  </body>
</html>
```

- This is how to get started and install tailwindcss in a plain html projects, if you want to know how to install tailwindcss in a react project follow [this](how-to-install-tailwindcss-in-react)
