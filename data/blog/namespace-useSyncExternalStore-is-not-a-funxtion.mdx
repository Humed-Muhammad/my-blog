## Introduction

**useSyncExternalStore** is a hook recommended for reading and subscribing from external data sources in a way that's compatible with concurrent rendering features like selective hydration and time slicing.

- Today we are going to see how to fix namespaces.useSyncExternalStore is not a function

## Syntax

```tsx
const state = useSyncExternalStore(subscribe, getSnapshot, getServerSnapshot?);
```

## The Isssue

- This isssue only occurs in the libraries that uses a react version >= 18+
- This hook was created to make external data sources reading compatible with concurrent rendering features like selective hydration and time slicing.

## How to fix it

- Simply upgrade your react to the latest version means 18+

```bash
yarn add react || npm i react
```

<Note>
  **Note:** By default both `yarn add react` **and** `npm i react` will install for you the latest
  version of react.
</Note>
