## Introduction

Node.js is an open-source, cross-platform back-end JavaScript runtime environment designed to build scalable network applications. It runs on a JavaScript Engine and executes JavaScript code outside of a web browser.

- Today we are going to see how we can install Node.js on ubuntu, thene with out further ado, let's get started.

## How to install Node.js on ubuntu

- There are several ways to install Nodejs on ubuntu, I will show you 2 methods, the second one is my favorite method.

## Using ppa

#### 1. By adding Node.js as ppa

- Install curl if you don't already have it installed.

```bash
sudo apt-get install curl
```

#### 2. Install Node.js

```bash
sudo apt-get install nodejs
```

#### 3. Verify

```bash
node -v
```

## Using NVM (node version manager)

- This part of the article will show you have to install Node.js using NVM (Node Version Manager)

#### 1. Install NVM

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
```

#### 2. Verify

```bash
nvm -v
```

#### 3. Install Node.js

- To install a specific version of Node.JS, specify the desired version number:

```bash
nvm install 18.8.0
```

- To install the latest release of node, use node, which is an alias for the latest version.

```bash
nvm install node
```

#### 4. Verify

```bash
node -v
```
